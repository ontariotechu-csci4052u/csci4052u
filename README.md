# csci4052u

A Python helper library for machine learning pipelines and model construction, built for CSCI 4052U.

## Installation

```bash
pip install git+https://github.com/ontariotechu-csci4052u/csci4052u
```

## Modules

- **data** — Data sources and transformations (e.g., `FashionMNISTDataModule`)
- **training** — Training loop helpers with TensorBoard logging, early stopping, and model checkpointing
- **report** — Reporting and inspection of training logs

## Dependencies

- PyTorch
- TorchVision
- Lightning
- TensorBoard
